# 🧾 Store Policies — Public Change Log

Welcome!  
This page is a **public record** of our store policies and their updates. We use **GitHub**, a trusted, third-party platform, to make sure every change is **transparent, and timestamped**.

---

## 🔍 What is GitHub?
GitHub is a platform that stores files and tracks every change made to them, kind of like a public “edit history.”  
Each update (called a **commit**) automatically shows:
- **Who made the change**
- **When the change was made**
- **Exactly what was changed**

You don’t need a GitHub account to view anything; just click through the files or the changelog below.

---

## 📅 How to See Policy Changes
1. **Click on any policy file** (for example, `privacy.md` or `shipping-returns.md`) to read the latest version.  
2. Near the top right, click **“History”** to see every edit made to that file.  
   - You’ll see **dates**, **authors**, and the **specific lines changed**.  
3. You can click any entry to view what changed before and after that date.

---

## 🧩 The CHANGELOG File
We also maintain a file called [`CHANGELOG.md`](./CHANGELOG.md).  
This lists every update we’ve made, newest first, with the **date** and a short note about what changed.  
For example:
```
## 2025-10-09 — Privacy Policy
- Clarified how we store and protect customer data.
```

---

## 🤝 Why We Do This
Our goal is simple: **trust through transparency.**  
By keeping this record public and verifiable, customers can:
- Confirm when any policy was last changed  
- See exactly what was modified  
- Be confident that no policy is quietly edited without a visible history
